# Bismi Chicken Shop Management System

## Project Overview
The Bismi Chicken Shop Management System is a comprehensive web application designed to streamline all aspects of managing a chicken shop business. It provides tools for inventory management, order processing, customer relationships, supplier management, financial transactions, and business reporting.

## Key Features

### Dashboard
- Real-time overview of business metrics
- Sales charts by day, week, and month
- Low stock alerts
- Recent orders
- Pending supplier payments

### Inventory Management
- Track chicken products and other inventory items
- Monitor stock levels
- Record item costs and selling prices
- Set reorder points

### Order Management
- Create and process customer orders
- Track order status (pending, completed, cancelled)
- Calculate order totals
- Record customer details

### Customer Management
- Store customer information including hotels
- Track pending balances
- View order history
- Process payments

### Supplier Management
- Maintain supplier details
- Track outstanding balances
- Record purchase history
- Process payments

### Transactions
- Record all financial transactions
- Categorize by type (payment, receipt, expense)
- Filter by date range
- Connect to customers and suppliers

### Reports
- Generate sales reports by date range
- View profit/loss statements
- Analyze customer and supplier activity
- Export data for accounting purposes

## Technical Implementation

### Frontend
- React for UI components
- TanStack Query for data fetching and state management
- Shadcn UI components for consistent design
- Responsive layout for mobile and desktop
- Charts.js for data visualization

### Backend
- Express.js server
- Firebase Realtime Database for data storage
- RESTful API architecture
- TypeScript for type safety

### Deployment
- Vercel for hosting
- Firebase for database
- Environment variables for configuration

## Getting Started
1. Clone the repository
2. Install dependencies with `npm install`
3. Configure environment variables for Firebase
4. Start the development server with `npm run dev`
5. For production deployment, follow instructions in VERCEL_DEPLOYMENT.md
